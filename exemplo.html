Comandos

Comandos básicos

  • ls -> lista os diretórios
  • ls -l -> lista os o que tem no diretório com mais informações em forma de lista.
  • touch -> cria um arquivo
  • mkdir -> cria diretórios (pastas)
  • cp -> copia um arquivo para outro local
  • cp -r -> copia uma pasta com todo o conteúdo para outro local
  • mv -> move um arquivo ou diretório para outro local.
  • rm -> remove um arquivo
  • rm -r -> remove uma pasta
  • cd -> entra em um diretório
  • cd .. -> volta um diretório
  • clear -> limpa o terminal

Atualizar o ubuntu

  • sudo apt update -> verifica se existe atualizações
  • sudo apt upgrade -> instala as atualizações
